# Contributing to DevOps Interview Questions

Thank you for contributing! Here's how to add new content.

## Adding a New Tool or Topic

1. Create a new folder under `topics/` with the next number:
   ```
   topics/28-your-new-topic/
   ```

2. Add two files:
   - `theory.md` — Theory-based questions and answers
   - `scenarios.md` — Scenario-based questions and answers

3. Follow the existing format:
   - Use `## Q1:` for questions
   - Use `**Answer:**` for answers
   - Include code snippets in fenced code blocks
   - Keep language simple and jargon-free

4. Update `README.md` to include your new topic in the Table of Contents.

5. Submit a pull request with a clear description.

## Guidelines

- Keep answers concise but complete
- Explain every technical term on first use
- Include practical examples and commands
- Make sure answers reflect current (2026) best practices
- Use consistent markdown formatting
