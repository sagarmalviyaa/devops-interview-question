## 🎯 How to Use This Repository

1. **Start with the [Study Guide](./getting-started/study-guide.md)** to understand the recommended learning order.
2. **Follow the numbered topics** — they're arranged from foundational to advanced.
3. **Read theory first**, then practice with scenarios for each topic.
4. **Use the [Interview Tips](./getting-started/interview-tips.md)** to maximize your preparation.
5. **Bookmark and revisit** — spaced repetition helps you remember.
